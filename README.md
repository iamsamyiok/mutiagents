# 微工作流引擎

> 一个简单、轻量、好用的多智能体协同工作流引擎

## ✨ 特色

- **简单** - 无需复杂配置，开箱即用，5分钟上手
- **轻量** - 单文件部署，无依赖安装，资源占用极低
- **好用** - 可视化编排，拖拽设计，所见即所得

## 🚀 快速开始

### 前端使用

1. 下载 `index.html` 文件
2. 用浏览器打开即可使用
3. 配置 LLM API Key
4. 开始创建你的第一个工作流

### 服务器同步（可选）

```bash
# 安装依赖
npm install

# 启动服务器
npm start
```

## 🔧 配置说明

### LLM API 配置

点击右上角设置图标，配置以下信息：

- **API 提供商**：OpenAI、Claude、或兼容接口
- **API Key**：你的 API 密钥
- **基础 URL**：API 地址（可选）
- **模型名称**：要使用的模型

### 知识库配置

- **嵌入 API**：用于向量化的 API 配置
- **相似度阈值**：检索匹配的最低相似度
- **返回数量**：每次检索返回的结果数

## 📝 常见问题

### Q: 需要安装什么软件吗？

A: 前端使用完全不需要安装，直接用浏览器打开 `index.html` 即可。如果需要多设备同步，可以运行可选的服务器组件。

### Q: 支持哪些 LLM 模型？

A: 支持所有兼容 OpenAI API 格式的模型，包括 GPT-4、Claude、Llama 等各种开源模型。

### Q: 数据保存在哪里？

A: 默认保存在浏览器本地存储中，也可以导出备份或使用服务器同步功能。

### Q: 可以离线使用吗？

A: 可以，工作流设计和本地功能完全离线可用。只有调用 LLM API 时需要网络连接。
